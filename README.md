## :toolbox: Getting Started

1. Make sure **Git** and **NodeJS** are installed.
2. Clone this repository to your local machine.
3. Install project dependencies using `npm install --legacy-peer-deps` or `yarn install --legacy-peer-deps`
4. The application is now fully set up üëç and you can start it using `npm run dev` or `yarn dev`.

**NOTE:** Please make sure to keep API keys and configuration values secure and do not expose them publicly.
